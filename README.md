Sistema de Navegación Aérea - BFS
Proyecto de Inteligencia Artificial para la materia de **Ingeniería de Sistemas**.

Arquitectura de Solución
Este proyecto utiliza un algoritmo de **Búsqueda en Anchura (BFS)** para encontrar la ruta con menos escalas entre dos ciudades de México.

### Especificaciones Técnicas:
* **Lógica Base:** Implementada en Python (`appvuelos.py`) usando la clase `Nodo`.
* **Despliegue Visual:** Interfaz web en Modo Oscuro con visualización de la expansión del árbol.
* **Complejidad Espacial:** $O(b^d)$, donde $b$ es el factor de ramificación y $d$ la profundidad.
